# 单通道推荐 User Skills（全局索引）

本文件与同级 `*.md` 会安装到 **`~/.claude/skills/`**，供「本地代码助手」枚举；**实际运行能力**取决于你是否已安装对应 Skill（见仓库 `docs/TOOLS.md`）。

场景套餐见 `docs/SKILLS_INTEGRATION.md` §11。

**自主学习**：同级 **`self_learning.md`** 会安装到 `~/.claude/skills/`，供任务链跑完后桌面端自动复盘，及 Cursor `stop` 钩子引用；契约见该文件正文与 **`docs/Claude-Code最优方案-整合版.md`** **§2.8**。
